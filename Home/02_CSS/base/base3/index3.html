Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Располагаться слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сдклать отступ снизу от соседней равным 2em, при помощи свойства margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделать отступ снизу от соседней равным 2em, при помощи свойства margin-
bottom. КАртинка вставлена при помощи тега, плавание организованопо правому краю внутри
данной статьи для данной картинки.
